## FAQ

<details open>
  <summary>
    What is the difference between bastion and a coding agent like Opencode?
  </summary>
  <p>
    Opencode is one of many primitives for running an AI coding agent in your
    local machine. Bastion is an orchestration tool to replicate your AI coding
    agent and dev environment into many isolated instances that can operate in
    parallel.
  </p>
</details>

<details open>
  <summary>Why VMs and not just use git worktrees?</summary>
  <p>
    Git worktrees isolate source code. They do not isolate everything an agent
    touches while working: ports, background processes, package installs,
    databases, temporary files, credentials, and local services. Bastion gives
    each coding agent its own runtime so agents can build, test, and modify code
    without colliding with one another.
  </p>
</details>

<details open>
  <summary>
    Is this compatible with my coding agent and dev environment?
  </summary>
  <p>
    At its core, a Bastion environment is backed by a Cloud Hypervisor VM. They
    are capable of supporting any dev environment that runs on Linux. You can
    configure these environments through a growing ecosystem of preset actions,
    create your own reusable setup, or run arbitrary shell commands.
  </p>
</details>
