<Steps>

1.  Define your agent's environment with a JSON template.

    ```json title="template.json"
    {
      "agents": {
        "opencode": {
          "working_directory": "/workspace/app",
          "auth": {
            "openai": {
              "type": "api",
              "key": "${{ secret.OPENAI_API_KEY }}"
            }
          },
          "config": {
            "model": "openai/gpt-5.5",
            "permission": "allow"
          }
        }
      },
      "actions": {
        "init": [
          {
            "use": "setup_bun"
          },
          {
            "run": "git clone https://github.com/.../app.git",
            "working_directory": "/workspace"
          },
          {
            "run": "bun install",
            "working_directory": "/workspace/app"
          }
        ]
      }
    }
    ```

2.  Initialize the template.

    ```sh
    bastion templates create --key app --file ./template.json
    # {
    #   "id": "tpl_123",
    #   "key": "app",
    #   "baseContentAddress": "sha256:...",
    #   "createdAt": "..."
    # }
    ```

3.  Deploy your agent.

    ```sh
    bastion env create --template-key app --key agent-1 --tag issue-32
    # {
    #   "id": "env_abc",
    #   "key": "agent-1",
    #   "status": "running",
    #   "templateId": "tpl_123",
    #   "tags": ["issue-32"],
    #   "createdAt": "...",
    #   "updatedAt": "..."
    # }
    ```

4.  Interface directly via tmux, OpenCode, or SSH.

        <Tabs>

      <TabItem label="tmux">

          ```sh
          bastion mux
          ```

        </TabItem>

      <TabItem label="OpenCode">

            ```sh
            bastion opencode --key agent-1
            ```

          </TabItem>

      <TabItem label="SSH">

            ```sh
            bastion ssh --key agent-1
            ```

          </TabItem>

    </Tabs>

</Steps>
